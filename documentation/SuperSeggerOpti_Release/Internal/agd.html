agd

PURPOSE

agd : autogains the image setting hot pixels to min value

SYNOPSIS

function im = agd( im ,HOT_PX)

DESCRIPTION 

 agd : autogains the image setting hot pixels to min value
 it sets pixels > hot pixels * Std.Dev to the min value

 INPUT : 
       im : image
       HOT_PX : hot pixel value
 OUTPUT :
       im : autogained image with increased contrast.
